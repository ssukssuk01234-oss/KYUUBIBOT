# Kyuubi Bot

**Kyuubi** is a Python Discord bot developed for moderation, automation, and custom server features.  
Developed and secured by _.kyuubi._.

## Features

- Connects to Discord via `discord.py`
- Console-based commands for management
- Supports DM interactions
- Logs bot activity in real-time
- Easy to deploy on Railway, Heroku, or local machine

## Requirements

- Python 3.13+
- `discord.py`
- `colorama`
- `aiohttp`

Install dependencies:

```bash
pip install -r requirements.txt
